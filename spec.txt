- 1: This corresponds to the actual line in the CSV file, not counting the header line. For cells 31 and 32, display "VFO1" and "VFO2" but grey out the cell.
- If a link is blank, display the cell number but nothing else.
- W: If the bandwidth is 12500, output "N". If it is 25000, output "W".

- K9IIK 2m: Display the actual title corresponding to the row.
